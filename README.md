# RAVE — Transfert de Timbre Audio

Application mobile React Native / Expo permettant de transformer le timbre d'un enregistrement audio grâce au modèle de réseau de neurones **RAVE** (VAE).

L'utilisateur enregistre ou sélectionne un clip audio, l'envoie à un serveur Python local, puis récupère et écoute le résultat transformé selon le modèle choisi (Jazz, Darbouka, Parole, Chats, Chiens).

## Lancer l'application

```bash
npx expo start
```

Scanner le QR code avec **Expo Go** sur iOS ou Android.
